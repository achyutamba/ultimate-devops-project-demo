# Ultimate DevOps Project — Documentation Hub

This `docs/` folder contains comprehensive, end-to-end documentation for the "ultimate-devops-project-demo" — an opinionated Azure-based microservices reference with Terraform, AKS, Helm, observability, security, and CI/CD.

Use this as the single source of truth for architecture, deployment, and operations.

Contents
- ARCHITECTURE.md — system architecture, diagrams, component responsibilities
- TERRAFORM.md — Terraform layout, modules, backend, variables, best practices
- HELM_K8S.md — Helm chart structure, values, templates, network policies, secrets
- CI_CD.md — Azure DevOps CI/CD design, service connections, pipeline stages
- OBSERVABILITY.md — OpenTelemetry, Prometheus, Grafana, Azure Monitor
- SECURITY.md — Key Vault, RBAC, network policies, scan pipeline
- DEPLOYMENT_RUNBOOK.md — step-by-step deploy and rollback runbooks
- TROUBLESHOOTING.md — common errors and fixes
- QUICKSTART.md — minimal steps to get dev environment running
 - GITOPS_ARGOCD.md — Argo CD GitOps deployment model and migration strategy
